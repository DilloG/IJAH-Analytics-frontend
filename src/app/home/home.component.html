What is IJAH Analytics?

Indonesian Herbal Herbs Analytics or IJAH Analytics is a website that provide prediction about herbal formulas that are suitable for a disease.

IJAH Analytics is inspired by the habits of Indonesian people who use medicinal plants as a treatment for maintaining health, whose properties are known for generations.

Our Feature

Drug-Target Connectivity

The drug targeting formulation focuses on the types of plants that contain compounds and then targets proteins that have a role in a disease. using input from the drug side or target side including plants, compounds, proteins, and diseases, using k-partite graph visualization as a result of input output. so the user can Validate / cross check whether some plant compositions, namely the Herbal formula, have certain properties

Classification Connectivity

Molecular new herbal formulations are produced by focusing on compounds that target certain diseases. Through the four properties of plants that are the basic ingredients of herbal medicine with the main properties which targets certain diseases, with compound input with four properties, analgesics, anti-inflammatory, antibacterial, and compounds with activity certain efficacy. The data used is taken from the IJAHAnalytics database

Compound synergy

Analysis of the synergistic effect on the constituents of drugs needs to be done to regulating the obtained herbal formula has a positive synergy effect from the composition compiler of herbal formula. General Effects done conventionally in the laboratory case by case.
